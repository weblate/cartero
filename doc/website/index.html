Cartero

Make HTTP requests and test APIs

Also available in other package managers.

Available as a Nix flake and through Homebrew for macOS.

Download for your OS:

  • Windows
    • x64, installer
    • x64, portable
  • macOS
    • Apple Sillicon
    • Intel 64-bit
  • AppImage
    • amd64

Cartero is free software released as GNU GPL 3.0 or later. The source code is available here.

For development updates, you can follow Cartero on Mastodon and other social networks of the fediverse (also bridging to Bluesky).

Test your APIs in a cozy environment

Cartero aims to be graphical and clean. Eventually packed with features, but without cluttering your view so that you can get your job done quickly.

Note: Cartero is still under development and not every feature is implemented yet! Check out our kanban to see what's on the menu for v0.2.0. You might want to follow the project on GitHub to get notified about new releases if you don't install the app from Flathub or AUR.
